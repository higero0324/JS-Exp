今回紹介する技術

今回はJavaScriptの技術を紹介したいと思います。

JavaScriptの技術は大まかに以下に分類されます。

  • フロントエンド

    • ユーザーが直接触れる部分を作る技術で、画面表示・操作・動き・デザインなどが対象となっており、ブラウザ上で動作します。

  • バックエンド

    • サーバーサイドで動作する部分を作る技術で、データベースとの連携やAPIの実装などが対象となります。

  • ツール

    • 開発を支援するためのツールやライブラリを作る技術で、ビルドツールやテストフレームワークなどが含まれます。

今回紹介するのは、中でもフロントエンドに分類される、Three.jsです。

Three.jsってなに？

Three.jsは、Webブラウザ上で本格的な3D表現を可能にするJavaScriptライブラリです。

  • HTMLと組み合わせて簡単に表示できる
  • 3Dモデルやアニメーションも扱える
  • 無料で商用利用も可能

📦 デモ：立方体を表示してみよう！

サンプルコード



// Three.js基本デモ：回転する立方体
const container = document.getElementById("canvas-container");

// シーン、カメラ、レンダラー
const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(75, container.clientWidth / container.clientHeight, 0.1, 1000);
const renderer = new THREE.WebGLRenderer({ antialias: true });
renderer.setSize(container.clientWidth, container.clientHeight);
container.appendChild(renderer.domElement);

// オブジェクト作成
const geometry = new THREE.BoxGeometry();
const material = new THREE.MeshStandardMaterial({ color: 0xffff00 });
const cube = new THREE.Mesh(geometry, material);
scene.add(cube);

// ライト追加
const light = new THREE.DirectionalLight(0xffffff, 1);
light.position.set(1, 1, 2).normalize();
scene.add(light);

// カメラ位置
camera.position.z = 3;

// アニメーション
function animate() {
  requestAnimationFrame(animate);
  cube.rotation.x += 0.01;
  cube.rotation.y += 0.01;
  renderer.render(scene, camera);
}
animate();

// ウィンドウサイズ対応
window.addEventListener("resize", () => {
  camera.aspect = container.clientWidth / container.clientHeight;
  camera.updateProjectionMatrix();
  renderer.setSize(container.clientWidth, container.clientHeight);
});
